# KT Prep Trainer

Учебный frontend-проект для подготовки к КТ по четырем предметам:

- ТГО
- Английский язык
- Базы данных
- Алгоритмы и структуры данных

Стек: React, TypeScript, Vite, CSS, Node.js и PostgreSQL.

## Запуск

```bash
npm install
cp .env.example .env
# Вставьте PostgreSQL connection string из Neon в DATABASE_URL
npm run dev
```

В production выполните `npm run build`, затем `npm start`. Сервер автоматически
создаст таблицы, раздаст собранный frontend и будет хранить вопросы и результаты
в PostgreSQL. Для Neon используется переменная окружения `DATABASE_URL`.

## Возможности

- демо-вопросы по каждому предмету;
- добавление одного вопроса через форму;
- массовое добавление вопросов JSON-массивом;
- варианты ответов от `A` до `I`;
- несколько правильных ответов через `correctAnswers`;
- проверка JSON и понятные ошибки валидации;
- просмотр, поиск, редактирование и удаление вопросов;
- тест по отдельному предмету;
- режим реального КТ с настройками количества вопросов;
- хранение вопросов и статистики в PostgreSQL.

## Формат JSON

```json
[
  {
    "subject": "databases",
    "question": "Какие утверждения верны для первичного ключа?",
    "options": {
      "A": "Уникально идентифицирует запись",
      "B": "Может использоваться для связей",
      "H": "Всегда является командой SQL"
    },
    "correctAnswers": ["A", "B"],
    "explanation": "Первичный ключ уникален и часто используется во внешних связях."
  }
]
```

Старый формат `"correctAnswer": "A"` тоже поддерживается при импорте.
